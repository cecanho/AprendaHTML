Um elemento HTML inicia um parágrafo em uma nova linha.

Enquanto estiver escrevendo o texto dentro das tags do parágrafo, o navegador entenderá que se trata de um único parágrafo.
Entretanto, com uma quebra de linha posso dividir este parágrafo. 

		O texto abaixo é criado em várias linhas com a tag pre:
