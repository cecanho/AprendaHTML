Todos os elementos devem ser fechados

Iníciou um parágrafo o mesmo deve ser fechado: <p> <p/>.
Ao utilizar quebra de linha (pular uma linha) o mesmo de ser fechado: <br />.

Tags, atributos de tags devem ser escritos com letra minúscula

<p font="font-family: arial">Olá Mundo</p>

Os valores dos atributos da tag devem estar entre aspas

<p font="font-family: arial">Olá Mundo</p>

A minimização de atributos de tags deve ser esquecida
