Exemplo de uma página dentro de outra

O que é iFrame?

Basicamente, o iFrame é uma tag ou etiqueta em um documento HTML que viabiliza a inclusão de outro documento HTML dentro do primeiro. Em outras palavras, ele permite que, a partir de uma página inicial, seja possível abrir uma seção de outra página qualquer.
